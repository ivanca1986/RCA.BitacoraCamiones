# Requerimientos no funcionales

## RNF-01: Disponibilidad offline
El sistema debe funcionar completamente sin conexión a internet.

## RNF-02: Rendimiento
El sistema debe responder en menos de 2 segundos en operaciones locales.

## RNF-03: Seguridad
Los datos almacenados deben estar protegidos en el dispositivo.

## RNF-04: Escalabilidad
El sistema debe permitir integración futura con módulos como facturación y control de empleados.

## RNF-05: Usabilidad
La interfaz debe ser simple e intuitiva para el usuario.
