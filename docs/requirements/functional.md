# Requerimientos funcionales

## RF-01: Registro de ingreso
El sistema debe permitir registrar el ingreso de un camión con:
- Placa
- Conductor
- Fecha y hora automática

## RF-02: Captura de fotografías
El sistema debe permitir tomar fotografías del camión durante el ingreso.

## RF-03: Almacenamiento offline
El sistema debe guardar la información localmente si no hay conexión.

## RF-04: Sincronización de datos
El sistema debe sincronizar automáticamente o manualmente los registros pendientes cuando haya conexión.

## RF-05: Consulta de registros
El usuario debe poder visualizar los ingresos registrados.

## RF-06: Estado de sincronización
El sistema debe mostrar si un registro está:
- Pendiente
- Sincronizado
- Error
