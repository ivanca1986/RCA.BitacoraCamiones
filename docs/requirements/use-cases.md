# Casos de uso

## CU-01: Registrar ingreso

Actor: Usuario

Flujo:
1. Usuario abre app
2. Ingresa datos del camión
3. Toma fotos
4. Guarda registro

Resultado:
Ingreso almacenado localmente

---

## CU-02: Sincronizar datos

Actor: Sistema

Flujo:
1. Detecta conexión
2. Envía registros pendientes
3. Actualiza estado

Resultado:
Datos sincronizados con la nube
